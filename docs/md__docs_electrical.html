Assembly Notes

Required Hardware Modifications

The following hardware modifications are required before/during assembling each reader unit.

  • Remove LoRa IC (SX1278) from ESP32 TTGO dev board:
    1. Remove OLED screen screws
    2. Use hot air / reflow tool to remove the SX1278 IC from the board and clean the traces
    3. Replace the OLED screen screws
  • Disable pin 12 for Y-axis input GPIO, use pin 14 instead:
    1. When soldering the male header pins to the ESP32 TTGO dev board, cut/remove the header pin for pin 12
    2. Make a solder bridge on the bottom of the PCB, shorting pin 12 and pin 14.
  • Disable automatic card read beep on new RFID readers:
    1. Obtain the HID Global configuration card from the ACM parts kit
    2. Power off the new RFID reader unit
    3. Place the configuration card within scanning range of the reader (still off)
    4. Apply power to the system while holding the card in position, and wait for the RFID reader to begin flashing red/purple (may require several seconds).
