137 /*
138  * @brief Create a new non-blocking TLS/SSL connection
139  *
140  * This function initiates a non-blocking TLS/SSL connection with the specified host, but due to
141  * its non-blocking nature, it doesn't wait for the connection to get established.
142  *
143  * @param[in] hostname Hostname of the host.
144  * @param[in] hostlen Length of hostname.
145  * @param[in] port Port number of the host.
146  * @param[in] cfg TLS configuration as esp_tls_cfg_t. `non_block` member of
147  * this structure should be set to be true.
148  * @param[in] tls pointer to esp-tls as esp-tls handle.
149  *
150  * @return - 1 If connection establishment fails.
151  * - 0 If connection establishment is in progress.
152  * - 1 If connection establishment is successful.
