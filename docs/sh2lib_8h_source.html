20 /*
21  * This is a thin API wrapper over nghttp2 that offers simplified APIs for usage
22  * in the application. The intention of this wrapper is to act as a stepping
23  * stone to quickly get started with using the HTTP/2 client. Since the focus is
24  * on simplicity, not all the features of HTTP/2 are supported through this
25  * wrapper. Once you are fairly comfortable with nghttp2, feel free to directly
26  * use nghttp2 APIs or make changes to sh2lib.c for realising your objectives.
27  *
28  * TODO:
29  * - Allowing to query response code, content-type etc in the receive callback
30  * - A simple function for per-stream header callback
31  */
