function getActivitySummaryByUser (   range,
  batchSize,
  filter 
)

Given a specific Range from an "Activity" sheet, load the data from the sheet in batches and summarize the usage by MakerLabs ID. Supports optional set of filter values which each row must match to be included.

Parameters
rangeRange of "Activity" sheet rows to be summarized.
batchSizeNumber of rows to fetch in each batch, to avoid loading entire sheet into memory.
filter(optional) Object with column titles as keys, and expected values as values. Rows with any columns that do not match their filter values are ignored.
Returns
Object containing activity summary by MakerLabs ID.

(Extract the header rows)

(Verify the expected columns were found, otherwise exit early)

  • Iterate through the rows in batches
  • (Adjust size if outside bounds)
  • Iterate through the batch rows
  • Only consider rows with a MakerLabs ID set

(If filter is set, ensure that all key/values have matching values in this row)

(If this is the first row for a user, populate the User columns that do not change)

(If this is the first activity for this machine ID, create an empty array)

  • Start an empty activity array with the row for referencing later
  • Iterate through this row's columns
  • Append the activity
  • Return the collected activity summaries.
