ImageFileUploader::firstUpdated (   changedProperties)
inline

LitElement lifecycle hook. Called after the element's DOM has been updated the first time, immediately before updated() is called.

Parameters
changedPropertiesJS Map containing properties which have changed

Setup event listeners:

  • On dragover, draw a border around the drop zone.
  • On dragleave, cancel the border when no longer dragged over.
  • On drop, begin the file upload immediately.
  • On change, begin the file upload after selecting a file with the chooser dialog
