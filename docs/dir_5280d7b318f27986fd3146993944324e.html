Detailed Description

Utilities and helpers using the Google Apps Script Spreadsheet APIs (e.g. Sheet, Range, NamedRange).

Files

file  excludeHeadersFromRange.js
 Given an arbitrary Range, return an adjusted range so that no header/frozen rows are included.
 
file  excludeTemplateRowFromRange.js
 Given an arbitrary Range, return an adjusted range so that no header/frozen rows or template row (first non-frozen row) is included.
 
file  getActivitySummaryByUser.js
 Given a specific Range from an "Activity" sheet, load the data from the sheet in batches and summarize the usage by MakerLabs ID. Supports optional set of filter values which each row must match to be included.
 
file  getDataRowsForSheet.js
 Given the name of a sheet in the Google Spreadsheet, return a Range which represents all non-header rows based on # of frozen rows.
 
file  getFirstEmptyRow.js
 Search the provided Range for the first row which has no data in all columns from the range, and return the index of that row.
 
file  getInstagramMediaUrl.js
 Replace any instagram.com URL with something like the following: https://www.instagram.com/p/<shortcode>/media/?size=m
 
file  getNamedRange.js
 Return the Range corresponding to the specified NamedRange name in a specific Google Spreadsheet sheet name.
 
file  getNextMakerLabsId.js
 Search through the "users_makerlabs_id" NamedRange in the "Users" sheet, parse the MakerLabs IDs (e.g. "ML123") in that column and return the next largest MakerLabs ID (e.g. "ML124").
 
file  getRangeIntersection.js
 Given two Ranges, return their intersection (the rows and columns they share in common).
 
file  getTemplateRowForSheet.js
 Given the name of a sheet within the Google Spreadsheet, return a Range which represents the template row (first non-frozen row).
 
file  getUsersFields.js
 Use NamedRanges on the "Users" sheet to determine the section title and column title for each column/field and include its associated metadata.
 
file  sendActivitySummaryEmail.js
 Send an HTML e-mail via Google Apps Script MailApp, using provided title, content and footer and generating content from templates/activity-usage-email.html. Update provided column index with current timestamp for all rows which were summarized in the e-mail.
 
file  sendBrandedEmail.js
 Send an HTML e-mail via Google Apps Script MailApp, using provided title, content and footer.
 
file  toColumnName.js
 Get column letter from column index.
 
file  toColumnNum.js
 Get column number from column letter.
